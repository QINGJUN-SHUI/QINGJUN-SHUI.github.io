# Research images

Use the following base names. Any of these formats are supported: `.png`, `.jpg`, `.jpeg`, `.webp`, `.avif`, `.gif`, `.svg`.

- `01-nir-oled`
- `02-multifunctional-diodes`
- `03-blue-uc-oled`
- `04-evaporable-fullerenes`
- `05-cofs`

Example: `01-nir-oled.png` or `01-nir-oled.jpg` both work.

Keep only one image per base name.
