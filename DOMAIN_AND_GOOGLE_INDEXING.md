# Google indexing and visibility notes

## Google Search vs Google Scholar

Google Search indexes normal web pages such as this homepage.
Google Scholar mainly indexes scholarly literature and author profiles; it is not a general personal-homepage search engine. A personal website may not appear inside Google Scholar search results even when it is accessible on the public web.

## Why the homepage may not appear in Google Search yet

Common reasons:

1. The homepage was created or changed recently, and Google has not crawled it again.
2. The site has very few external links pointing to it.
3. Google Search Console has not been set up.
4. The browser search query is too broad, for example only searching `Shui Qingjun`.

## Recommended actions

1. Open Google Search Console.
2. Add the property `https://qingjun-shui.github.io/`.
3. Verify ownership.
4. Submit the sitemap:

```text
https://qingjun-shui.github.io/sitemap.xml
```

5. Use URL Inspection for:

```text

https://qingjun-shui.github.io/publications/
https://qingjun-shui.github.io/research/
```

6. Click “Request indexing”.
